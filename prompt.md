# Профессиональный промпт для разработки сайта BYD на Next.js

## Общая информация о проекте
Создать современный веб-сайт автомобильного дилера BYD с использованием Next.js 14+ (App Router), TypeScript, Tailwind CSS и shadcn/ui. Сайт должен быть полностью адаптивным, оптимизированным для SEO и поддерживать 3 языка (RU, UZ, EN) с переключением светлой/темной темы.

## Технологический стек
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Slider**: Swiper.js или Embla Carousel
- **Forms**: React Hook Form + Zod validation
- **Internationalization**: next-intl
- **State Management**: Zustand (для языка и темы)

## Страницы и функционал

### 1. Главная страница (/)
**Hero Section**:
- Полноэкранный слайдер с 3-5 топовыми моделями
- Каждый слайд: фоновое изображение авто, название модели, краткое описание, цена "от X сум", кнопки "Подробнее" и "Тест-драйв"
- Анимированный текст и плавные переходы
- Индикаторы слайдов и стрелки навигации

**Секция "Популярные модели"**:
- Grid карточек (3-4 в ряд на desktop)
- Каждая карточка: изображение авто, название, краткие характеристики (двигатель, мощность), цена, кнопка "Подробнее"
- Hover эффекты: легкий подъем карточки, тень, увеличение изображения
- Используй shadcn/ui Card компонент

**Секция "Почему BYD"**:
- 4-6 карточек преимуществ с иконками
- Статистика с анимированными счетчиками (лет на рынке, проданных авто, моделей)

**Секция "Новости/События"**:
- Горизонтальный слайдер с новостями
- Карточки с изображением, датой, заголовком

**CTA секция**:
- Призыв к действию "Запишитесь на тест-драйв"
- Яркая кнопка, контрастный фон

### 2. Каталог моделей (/models)
**Фильтры (sidebar или top bar)**:
- Тип кузова (Седан, Кроссовер, Хэтчбек)
- Тип двигателя (Бензин, Гибрид, Электро)
- Ценовой диапазон (slider)
- Год выпуска
- Кнопки "Применить" и "Сбросить"

**Grid моделей**:
- Адаптивный grid (1-2-3-4 колонки)
- Карточки с hover эффектами
- Бейджи "Новинка", "Хит продаж", "Электромобиль"
- Quick view при наведении

### 3. Страница модели (/models/[slug])
**Hero секция**:
- Большой слайдер с изображениями авто (экстерьер, интерьер, детали)
- Галерея thumbnails внизу
- Fullscreen режим галереи

**Информационные блоки**:
- Название модели и цена (крупно)
- Табы: Обзор, Характеристики, Комплектации, Цвета
- **Обзор**: описание, ключевые особенности с иконками
- **Характеристики**: таблица (двигатель, мощность, разгон, расход, размеры)
- **Комплектации**: карточки комплектаций с ценами и возможностями
- **Цвета**: интерактивный выбор цвета кузова с изменением изображения авто

**Калькулятор кредита** (встроенный на странице):
- Цена авто (auto-filled)
- Первоначальный взнос (slider)
- Срок кредита (dropdown)
- Интерактивный расчет ежемесячного платежа
- Кнопка "Отправить заявку"

**CTA блоки**:
- "Записаться на тест-драйв" (кнопка открывает модальное окно)
- "Получить консультацию"
- Кнопки для скачивания брошюры (PDF)

**Похожие модели**:
- Слайдер с 3-4 похожими моделями

### 4. Тест-драйв (/test-drive)
**Hero с формой**:
- Заголовок и описание процесса
- Форма (shadcn/ui Form):
  - Выбор модели (dropdown с поиском)
  - Имя, телефон, email
  - Дата и время (date/time picker)
  - Комментарии (textarea)
  - Checkbox согласия с политикой
  - Кнопка "Записаться"
- Валидация с подсветкой ошибок
- Toast уведомление об успешной отправке

**Информационные блоки**:
- "Как проходит тест-драйв" (steps с иконками)
- FAQ accordion
- Контакты дилерских центров на карте

### 5. Сервис (/service)
**Услуги** (grid карточек):
- ТО, Ремонт, Диагностика, Кузовной ремонт, Запчасти
- Каждая карточка: иконка, название, описание, цена "от"

**Форма записи на сервис**:
- Аналогична форме тест-драйва
- Выбор типа услуги
- Выбор дилерского центра
- Описание проблемы

**Преимущества сервиса**:
- Оригинальные запчасти
- Квалифицированные специалисты
- Гарантия на работы

### 6. Кредитный калькулятор (/calculator)
**Интерактивный калькулятор**:
- Большая форма с визуализацией
- Стоимость автомобиля (input с autocomplete моделей)
- Первоначальный взнос (slider + input, показывать %)
- Срок кредита (slider, 6-60 месяцев)
- Результаты расчета (крупно):
  - Ежемесячный платеж
  - Переплата
  - Общая сумма
- График платежей (таблица или chart)
- Кнопка "Отправить заявку на кредит"

**Партнеры банки**:
- Логотипы банков-партнеров
- Условия кредитования

### 7. О компании (/about)
**Секции**:
- История BYD (timeline)
- Миссия и ценности
- Достижения (статистика)
- Команда (если применимо)
- Сертификаты и награды

### 8. Контакты (/contacts)
**Информация**:
- Адрес, телефон, email (clickable)
- Режим работы
- Интерактивная карта (Google Maps или Yandex Maps)
- Форма обратной связи

**Дилерские центры**:
- Список всех центров с адресами на карте
- Фильтр по городам

### 9. Страница "Оставить заявку" (модальное окно)
**Форма**:
- Имя, телефон, email
- Интересующая модель (optional)
- Комментарий
- Согласие на обработку данных
- Красивое modal окно (shadcn/ui Dialog)

## Дизайн-система

### Цветовая палитра (адаптировать из текущего сайта BYD)
**Light theme**:
- Primary: #C41E3A (красный BYD) или аналог
- Secondary: #1F2937 (темно-серый)
- Background: #FFFFFF, #F9FAFB
- Text: #111827, #6B7280

**Dark theme**:
- Primary: #DC2626 (яркий красный)
- Secondary: #4B5563
- Background: #111827, #1F2937
- Text: #F9FAFB, #D1D5DB

**Header background**: 
- Dark theme: #0F172A или #1E293B (всегда темный для белого лого)
- Light theme: #1F2937 или #374151 (темный даже в светлой теме)

### Типографика
- Headings: Inter или Montserrat (bold, 600-700)
- Body: Inter (regular, 400-500)
- Размеры: 12px-48px (responsive с clamp)

### Компоненты shadcn/ui
Использовать:
- Button (варианты: default, outline, ghost)
- Card
- Dialog/Modal
- Form (Input, Select, Textarea, Checkbox)
- Tabs
- Accordion
- Slider
- Carousel
- Sheet (для mobile menu)
- Toast
- DropdownMenu
- Badge

### Анимации
- Плавные переходы (transition-all duration-300)
- Hover эффекты на карточках и кнопках
- Fade-in при скролле (Intersection Observer)
- Skeleton loaders для загрузки контента

### Отступы и spacing
- Container: max-w-7xl, px-4 sm:px-6 lg:px-8
- Sections: py-16 md:py-24
- Cards gap: gap-6 md:gap-8
- Consistent spacing system (4, 8, 12, 16, 24, 32, 48, 64px)

## Технические требования

### Производительность
- Lazy loading изображений (Next.js Image)
- Code splitting по роутам
- Оптимизация шрифтов (next/font)
- Lighthouse score > 90

### SEO
- Мета-теги для каждой страницы
- Open Graph и Twitter cards
- Structured data (JSON-LD для автомобилей)
- Sitemap.xml и robots.txt
- Альтернативные языковые версии (hreflang)

### Адаптивность
- Mobile-first подход
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px), 2xl (1536px)
- Touch-friendly элементы на мобильных
- Hamburger menu на мобильных (Sheet component)

### Интернационализация
- Переключение языка без перезагрузки
- Сохранение выбранного языка (localStorage)
- RTL поддержка не требуется (RU, UZ, EN - LTR)
- Перевод всех статичных текстов
- Динамический контент с API (с поддержкой мультиязычности)

### Формы
- Client-side валидация (Zod schema)
- Показывать ошибки inline
- Disabled состояние кнопок при отправке
- Success/Error toast уведомления
- Маски для телефона (+998 XX XXX-XX-XX)

### Accessibility
- Semantic HTML
- ARIA labels где необходимо
- Keyboard navigation
- Focus states
- Alt texts для изображений
- Contrast ratio соответствует WCAG AA
